# aegis-crypto-engine
Real-time cryptocurrency monitoring dashboard with a custom logic engine for market volatility analysis and signal generation.

Overview 

A real-time cryptocurrency monitoring dashboard designed to demonstrate high-frequency data processing and direct DOM manipulation using jQuery. The application integrates with the Binance Public API to analyze market volatility through a custom-built logic engine.

Core Features

Logic Engine: Implements a weighted scoring algorithm to detect market signals based on price change and volume triggers.

Performance Optimization: Utilizes a direct DOM update strategy to ensure low-latency UI refreshes during high-frequency data streams.

Event Handling: Real-time logging of market activities and volatility alerts.

Technical Stack

Frontend: jQuery 3.7.1, Bootstrap 5

API: Binance Rest API

Visualization: Chart.js

Testing: QUnit / Jest

How to Run

Clone the repository.

Open index.html in any modern web browser.

No build process or dependencies required.
